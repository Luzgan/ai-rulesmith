# Understand Before Changing

- Read and understand existing code before modifying it
- Look for existing functions, utilities, and patterns to reuse
- Identify all affected files and assess the blast radius
- Outline implementation steps in dependency order
- Get user alignment on approach before starting
- If discoveries during implementation change the plan, pause and reassess
